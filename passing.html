Passing the 1–6 Seam

If server is in zone 5: middle back has deep seam and right back has short seam. If server is in zone 1: middle back passer has short and right back passer has deep

Passing the 5–6 Seam

If server is in zone 1- left back passer takes short seam and middle back passer takes deep seam. If server is in zone 5- left back passer takes deep seam and middle back passer takes short seam.
